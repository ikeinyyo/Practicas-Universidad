More information in http://javavis.sourceforge.net

Unzip in a directory. 

- Execute run.sh or run.bat depending on your operating system and enjoy!
- JavaVis is prepared for a Eclipse project. Just import the project into Eclipse.
- If you are using another platform or even from command line, you must bear this in mind:
	- All the libs in the lib directory must be included in the CLASSPATH
	- The main class is javavis.Gui
